# EE2026-Digital-Design-AY25-26Sem2
Labs for EE2026 Digital Design

## About the final project: 
Syntheractive is a polyphonic synthesiser that enables users to 
play and record music while incorporating a game mode to 
make practice more engaging. Users can choose between 
instrument and game modes. In instrument mode, users can 
play and record a song, with support for up to four 
simultaneous key presses and selectable waveforms for varied 
sound quality. In game mode, inspired by Piano Tiles, a recorded 
song is converted into falling notes that users must match in 
time. The system provides real-time feedback and a final score, 
allowing users to track progress and refine their performance in 
an interactive manner.
